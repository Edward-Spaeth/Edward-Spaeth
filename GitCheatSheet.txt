# GitHub Cheat Sheet
"ls -a" um hidden ordner mit ls anzuzeigen
"rm -rf .git" um hidden gitordner zu removen

"git init" im target ordner

"git add ." um diese Files für das committen zu adden
"git commit -m "Generic Commit Message" " für das commiten der geaddeten Files
"git remote add origin <URL DES REPOSITORIES>"


Personal Access Token als Passwort im Terminal! (GitHub User Icon --> Settings --> Unten Developer Settings --> Personal Access Tokens)
